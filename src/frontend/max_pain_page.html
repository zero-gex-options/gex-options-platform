Option Value at Each Strike Price ℹ️ Max Pain Visualization

Shows the total intrinsic value of all options (calls and puts) if SPY were to close at each strike price.

Green (Calls): Value retained by call holders
Red (Puts): Value retained by put holders

The strike with the lowest total bar height is the Max Pain point - where option holders would lose the most money.
